Image Formats

 

A List of all the image formats


Modes

 
IL_BMP
Microsoft bitmap .
IL_CHEAD
C Header.
IL_CUT
Dr. Halo .cut image.
IL_DCX
.dcx image.
IL_DDS
DirectDraw Surface image.
IL_DOOM
Doom texture.
IL_DOOM_FLAT
Doom flat (floor).
IL_GIF
Graphics Interchange Format file.
IL_HDR
RADIANCE High Dynamic Range Image.
IL_ICO
Microsoft icon (.ico).
IL_JASC_PAL
Load the file into the current image's palette as a Paint Shop Pro (Jasc) palette.
IL_JPG
Jpeg.
IL_LIF
Homeworld image.
IL_MDL
Half-Life model file (.mdl).
IL_MNG
Load a Multiple Network Graphics (.mng).
IL_PCD
Kodak PhotoCD image.
IL_PCX
.pcx Image.
IL_PIC
Softimage Pic image.
IL_PIX
Alias | Wavefront .pix file.
IL_PNG
Portable Network Graphics (.png) image.
IL_PNM
Portable AnyMap (.pbm, .pgm or .ppm).
IL_PSD
PhotoShop (.psd) file.
IL_PSP
Paint Shop Pro file.
IL_PXR
Pxrar (.pxr) file.
IL_RAW
Raw data with a 13-byte header.
IL_SGI
SGI (.bw, .rgb, .rgba or .sgi).
IL_TGA
TrueVision Targa.
IL_TIF
TIFF (.tif or .tiff) image.
IL_TYPE_UNKNOWN
Tells OpenIL to try to determine the type of image present in FileName, File or Lump.
IL_WAL
Quake .wal texture.
IL_XPM
.xpm file.
