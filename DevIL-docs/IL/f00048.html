ilPopAttrib

The ilPopAttrib function "pops" the last entry off the state stack into the current states.

ILvoid ilPopAttrib(
  ILvoid
);

Parameters

None

Remarks

ilPopAttrib pops the last pushed stack entry off the stack and copies the bits specified when pushed by ilPushAttrib to the previous set of states.

Error Codes

IL_STACK_UNDERFLOW - The stack was popped without being pushed by ilPushAttrib.
