ilRemoveLoad

The ilRemoveLoad function removes a registered loading function from OpenIL's registered functions list.

ILboolean ilRemoveLoad(
  char *Ext);

Parameters

Ext
Extension to remove (e.g. "tga").


Remarks

ilRemoveLoad removes a registered extension handler from the registered load functions list. Use this function when a new handler for an extension needs to be registered.

Error Codes

None

See Also

ilRegisterLoad, ilRemoveSave, ilRegisterSave
