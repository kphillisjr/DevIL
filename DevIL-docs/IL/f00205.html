ilResetRead, ilResetWrite

The ilResetRead and ilResetWrite functions resets the file callbacks to DevIL's default handlers.

ILvoid ilResetRead(
  ILvoid
);


ILvoid ilResetWrite(
  ILvoid
);

Parameters

None

Remarks

ilResetRead resets the reading functions set by ilSetRead. ilResetWrite resets the writing functions set by ilSetWrite.

Error Codes

None

See Also

ilSetRead, ilSetWrite
