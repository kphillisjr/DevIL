ilRegisterPal

 

The ilRegisterPal function registers the current image's palette.

 

ILvoid ilRegisterPal(
  ILvoid Pal, ILuint Size, ILenum Type );


Parameters

 
Pal
Pointer to palette to be copied to the current image
Size
Size of Pal in bytes
Type
Type of the palette.

IL_PAL_BGR24

IL_PAL_BGRA32
