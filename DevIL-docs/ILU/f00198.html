iluDeleteImage

The iluDeleteImage function deletes a single image name.

ILvoid iluDeleteImage(
  ILuint Id);

Parameters

Id
The image name to delete.


Remarks

iluDeleteImage is a convenience function to delete a single image instead of calling ilDeleteImages.

Error Codes

IL_INVALID_VALUE - Id was less than one.

See Also

ilDeleteImages, iluGenImage, ilGenImages
