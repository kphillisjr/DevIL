ilutSetInteger

The ilutSetInteger function sets the value of a selected mode.

 

ILvoid ilutSetInteger(
  ILenum Mode,
  ILint Param
);

Parameters

Mode
The mode value to be returned. The following values are accepted:

ILUT_MAXTEX_WIDTH
Not used yet.
ILUT_MAXTEX_HEIGHT
Not used yet.
ILUT_MAXTEX_DEPTH
Not used yet.
ILUT_D3D_MIPLEVELS
Please refer to ilutD3D8MipFunc.

Param
The value to set the mode with.

Remarks

ilutSetInteger is ilutGetInteger's counterpart.

Error Codes

IL_INVALID_ENUM - Mode was an invalid enum.
IL_INVALID_PARAM - Param was not in range.
