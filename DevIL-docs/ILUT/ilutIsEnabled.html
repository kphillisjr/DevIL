ilutIsEnabled

 

ilutIsEnabled returns whether the mode indicated by Mode is enabled.

 

ILboolean ilutIsEnabled(
  ILenum Mode );


Parameters

 
Mode
Indicates an OpenIL mode


Modes

 
ILUT_OPENGL_CONV
Whether ilut is allowed to use GL_RGB8 and GL_RGBA8 instead of GL_RGB8 or GL_RGBA8 (useful with nVidia cards).
ILUT_PALETTE_MODE
Not used right now.

Errors

 
ILUT_INVALID_ENUM
Mode was of an invalid value.
